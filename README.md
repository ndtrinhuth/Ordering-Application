Cách chạy chương trình :

Download code từ Github về.

Mở Dev-C++, chọn File > New > Project.

Chọn Console Application, tick vào C++ Project, đặt tên Ordering Application rồi nhấn OK.

Lưu project này vào ngay trong thư mục mà bạn vừa giải nén mã nguồn GitHub lúc nãy.

Xóa cái file main.cpp mặc định mà phần mềm vừa tự tạo.

Trên thanh menu, chọn Project > Add to Project (hoặc chuột phải vào tên Project chọn Add Files).

Quét chọn tất cả các file .h và .cpp từ file include và src rồi bấm Open.

Đảm bảo 2 file txt (products.txt và cart.txt) nằm chung chỗ với file .dev hoặc .cbp của Project.

Nhấn F11 để chạy.
